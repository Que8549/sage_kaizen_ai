# Sage Kaizen — Canonical Definition

## Formal Definition

**Sage Kaizen** is a modular, dual-mode, locally hosted AI architecture designed to embody wisdom, adaptability, and continuous improvement through disciplined engineering and iterative refinement.

It integrates:

- Dual LLM inference engines (“Fast Brain” + “Architect Brain”)
- Multi-GPU orchestration (RTX 5090 + RTX 5080)
- Raspberry Pi device agents (Voice-Pi + RGB-Pi)
- A 6-panel 64×64 RGB LED matrix cube
- Voice loop (STT → LLM → Tools → TTS)
- Retrieval-Augmented Generation (RAG)
- Self-documenting code generation

Sage Kaizen is not merely a chatbot — it is a **Persistent Local AI Architect and Physical-World Interface System**.

### Kaizen

From Japanese 改善:
- 改 *Kai* means “change” or “revision.”
- 善 *Zen* means “good,” “virtue,” or “goodness.”
- Meaning: **continuous improvement**

Within the system, Kaizen represents:

- Iterative GPU optimization
- Prompt refinement
- RAG evolution
- Modular scalability
- Benchmark-driven tuning
